Что такое замыкание внутри функций

-когда внутри функций есть обращение какой либо перменной, она сначало ищет внутри себя и после этого, если она не находит данную переменную, то идет искать выше - во внешних. Поэтому замыкание - это, функция вместе со всеми внешними переменными которые ей доступны. Кроmе всех действий, функция может венрнуть какое либо значение при помощи ключевого слова return

Function Declaration

функция, объявленная в основном потоке кода.

var f = function(параметры) { // тело функции};

Function Expression

функциональное выражение

.

Oбъявление функции в контексте какого-либо выражения, например присваивания.

function имя(параметры) {...}
